HITOBITO - 개인정보 처리방침

한국어 日本語
시행일: 2026년 5월 4일
주식회사 심심라운지(이하 '회사')가 운영하는 HITOBITO 서비스는 「개인정보 보호법」 및 일본 「個人情報保護法」에 따라 이용자의 개인정보를 보호하고 이와 관련한 고충을 신속하고 원활하게 처리할 수 있도록 하기 위하여 다음과 같이 개인정보 처리방침을 수립·공개합니다.
제1조 (수집하는 개인정보)
1. 회원가입 시
필수: 이메일 주소, 비밀번호, 닉네임, 국적, 성별, 생년월일
선택: 프로필 사진, 자기소개, 관심사, 여행 계획, SNS 아이디(Instagram, Facebook)
2. 신분증 인증 시 (선택)
신분증 이미지: OCR 처리 목적으로만 임시 수집, 처리 완료 즉시 삭제 (서버 미저장)
실명: 본인확인 대조 목적으로만 임시 수집, 처리 완료 즉시 삭제 (서버 미저장)
인증 결과 메타데이터만 보관: 이름 일치 여부(boolean), 생년월일 일치 여부(boolean), 신뢰도 점수
3. 휴대폰 본인인증 시
휴대폰 번호: 본인확인 및 중복 가입 방지 목적으로 수집
4. 서비스 이용 시 자동 수집
기기 정보 (푸시 알림 발송용 FCM 토큰)
5. 로컬 체험 서비스 이용 시
위치 정보: 장소 검색 및 지도 표시를 위해 주소, 위도·경도 좌표를 수집
제2조 (개인정보의 처리 목적)
회원 가입 및 관리
한·일 여행 매칭 서비스 제공
본인확인 및 연령 검증 (만 19세 이상)
채팅 및 알림 서비스 제공
서비스 개선 및 부정 이용 방지
휴대폰 본인인증 (SMS 인증 코드 발송)
제3조 (개인정보의 제3자 제공 및 처리 위탁)
회사는 원칙적으로 이용자의 개인정보를 제3자에게 제공하지 않습니다. 다만, 서비스 제공을 위해 아래와 같이 개인정보 처리를 위탁하고 있습니다:
1. 신분증 인증 (OCR 처리)
제공 대상:
Google Cloud Vision API (Google LLC, 미국)
제공 항목:
신분증 이미지 (처리 즉시 삭제)
제공 목적:
문자 인식(OCR)을 통한 본인확인
보유 기간:
처리 즉시 삭제 (Google 측도 API 처리 후 데이터 미보유)
2. 채팅 메시지 번역
제공 대상:
DeepL SE (독일) / Naver Cloud (대한민국, 예비용)
제공 항목:
채팅 메시지 원문
제공 목적:
한국어·일본어 자동 번역 서비스 제공
보유 기간:
API 처리 즉시 삭제 (번역 결과만 서버에 저장)
3. 데이터 호스팅 및 인프라
제공 대상:
Supabase Inc. (미국, AWS 인프라)
제공 항목:
서비스 이용에 필요한 모든 데이터 (회원 정보, 채팅 메시지 등)
제공 목적:
데이터베이스 호스팅 및 인증 서비스 제공
보유 기간:
회원 탈퇴 시까지 (탈퇴 후 30일 유예 기간 경과 시 삭제)
4. 푸시 알림
제공 대상:
Firebase Cloud Messaging (Google LLC, 미국)
제공 항목:
기기 토큰(FCM 토큰), 알림 메시지 내용
제공 목적:
푸시 알림 발송
보유 기간:
서비스 이용 기간 중 유지
5. 오류 모니터링
제공 대상:
Sentry (Functional Software Inc., 미국)
제공 항목:
오류 로그, 기기 정보, 앱 상태 정보 (개인 식별 정보 미포함)
제공 목적:
앱 오류 감지 및 서비스 안정성 개선
보유 기간:
Sentry 기본 보유 정책에 따름 (90일)
6. 휴대폰 본인인증 (SMS)
제공 대상:
Twilio Inc. (미국)
제공 항목:
휴대폰 번호
제공 목적:
SMS 인증 코드 발송을 통한 본인확인
보유 기간:
Twilio의 메시지 로그 보유 정책에 따름 (기본 30일)
7. 소셜 로그인 (카카오)
제공 대상:
Kakao Corp. (대한민국)
제공 항목:
이메일 주소, 프로필 정보 (카카오 계정 기반)
제공 목적:
카카오 계정을 통한 간편 로그인
보유 기간:
회원 탈퇴 시까지
8. 소셜 로그인 (LINE)
제공 대상:
LINE Corporation (일본)
제공 항목:
이메일 주소, 프로필 정보 (LINE 계정 기반)
제공 목적:
LINE 계정을 통한 간편 로그인
보유 기간:
회원 탈퇴 시까지
9. 위치 정보 및 지도 서비스
제공 대상:
Google Maps Platform (Google LLC, 미국)
제공 항목:
검색어(주소), 위치 좌표
제공 목적:
로컬 체험 장소 검색 및 지도 표시
보유 기간:
API 처리 즉시 삭제 (검색 결과 좌표만 서버에 저장)
제3-2조 (개인정보의 국외 이전)
회사는 서비스 제공을 위해 아래와 같이 개인정보를 국외로 이전하고 있습니다:
이전 국가:
미국 (Supabase, Google, Sentry, Twilio), 독일 (DeepL), 대한민국 (Kakao), 일본 (LINE)
이전 항목:
제3조에 기재된 각 서비스별 제공 항목과 동일
이전 목적:
제3조에 기재된 각 서비스별 제공 목적과 동일
이전 방법:
네트워크를 통한 암호화 전송 (HTTPS/TLS)
각 수탁업체는 해당 국가의 개인정보 보호 법률을 준수하며, 계약에 의해 제공받은 개인정보의 안전한 처리를 보장합니다.
제4조 (개인정보의 보유 및 이용 기간)
회원 정보: 회원 탈퇴 요청 후 30일 유예 기간 경과 시 영구 삭제 (유예 기간 내 로그인 시 삭제 취소 가능)
신분증 이미지 및 실명: 인증 처리 완료 즉시 삭제 (서버 미저장)
인증 결과 메타데이터: 회원 탈퇴 요청 후 30일 유예 기간 경과 시 삭제
채팅 메시지: 회원 탈퇴 요청 후 30일 유예 기간 경과 시 삭제
휴대폰 번호: 회원 탈퇴 요청 후 30일 유예 기간 경과 시 삭제 (마스킹 처리하여 저장)
제5조 (개인정보의 파기)
회사는 개인정보의 보유 기간이 경과하거나 처리 목적이 달성된 경우, 지체 없이 해당 개인정보를 파기합니다. 특히 신분증 인증 시 수집되는 이미지와 실명은 인증 목적 달성 즉시 메모리에서 삭제되며, 어떠한 저장 매체에도 기록되지 않습니다.
제6조 (이용자의 권리)
이용자는 언제든지 다음의 권리를 행사할 수 있습니다:
개인정보 열람 요구
개인정보 정정·삭제 요구
개인정보 처리 정지 요구
회원 탈퇴 (설정 > 서비스 탈퇴하기, 요청 후 30일 유예 기간 적용)
제7조 (안전성 확보 조치)
개인정보는 암호화하여 저장 및 전송합니다 (HTTPS/TLS)
비밀번호는 단방향 해시 처리합니다
신분증 이미지는 서버에 저장하지 않으며, 인증 처리 후 즉시 메모리에서 삭제합니다
접근 권한을 최소화하고, 접근 기록을 관리합니다
